HOW TO DEPLOY REACTJS TO FIREBASE

Step 1: Access to firebase and sign in by account google.
Click Go to console or Get started


Select Add project



Enter the name of your project and do step as picture below

Step 2: After Create project you need you need to create your domain

You can access this link to create domain Freenom and Sign in


After entering the domain name you need to check if the domain name exists or not

Tip: You you can create domain name with .tk extension

If the domain name is available, you can choose to Checkout

Don't forget to choose the number of months, you can use it up to 12 months for free

After choose the number of months, click continue
Step 3:Open cmd, import npm install -g firebase-tool import firebase login



To be able to access your project you need to follow the pictures below



Import firebase init



? Are you ready to proceed?(Y/n) Import Y



Press ↓ Hosting and press Spacebar to select, finally press Enter




Select Use an existing project and press Enter



Choice your project and press Enter



Do steps as picture below


Step 4: Open firebase.json and add headers command into it



Step 5: Run command: firebase deploy



Wait for the command to finish running, there will be 2 links for you, in which Hosting URL is the path to Your ReactJs app. I get the page like belowe:
