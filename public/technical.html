    HOW TO DEPLOY REACTJS TO FIREBASE

    STEP 1: login firebase by google account. Then click go to console or get started

    - 1.1: click add project and do steps as picture below

    - Time to open VISUAL STUDIO CODE now!

    STEP 2: open Terminal and run: create-react-app nameProject

    - TIP: make sure that folder has nothing inside before create react app

    - 2.1: run npm i -g firebase-tools in terminal command

    - 2.2: run npm run build in terminal command

    STEP 3: using command login firebase to log in firebase

    STEP 4: run command firebase init in terminal command

    ? Are you ready to proceed? Y

    - 4.1: choose Hosting: Configure and deploy Firebase Hosting sites

    - NOTE: enter Space first then Enter, if something swrong, please run firebase init again

    - 4.2: choose Use an existing project

    - 4.3: answer question correctly as picture below

    - RESULT STEP 4: react app appears two new files

    STEP 5: open file firebase.json and add headers command into it

    FINAL STEP: run firebase deployand enjoy your achievement
